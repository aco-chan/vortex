# きもちの絵本

感情を選んでいくと、その気持ちを否定せずに受けとめる、絵本風の短い物語を表示するiPhone向けWebモックです。

## 入っているもの

- タイトル画面
- R: Raw Emotion として、いまの気持ちをそのまま選ぶ画面
- 主人公の名前と呼び方を選ぶ画面
- L: Labeling として、気持ちの奥にありそうな願いを選ぶ画面
- T: Translation として、気持ちを景色や物語に翻訳する画面
- R': Refined Reaction として、新しい見方と「今日のコンパス」を添えた絵本風ストーリー出力画面

## 考え方

このモックは、感情を「直す」ためのものではありません。
怒り、涙、不安、疲れ、もやもやを、悪いものとして追い出すのではなく、
その奥にある願いや大切にしたいものに気づくための小さな物語として出力します。

流れは `R -> L -> T -> R'` を参考にしています。

- `R`: まず、そのままの感情を選ぶ
- `L`: 感情の奥にある言葉を選ぶ
- `T`: ぴーとごまじろうが、感情を景色や比喩に翻訳する
- `R'`: 解決ではなく、少し違う視点で自分の気持ちを見られるようにする

主人公の名前と呼び方を選ぶと、最後の物語に反映されます。
ネガティブ寄りの気持ちだけでなく、うれしい、楽しい、安心、誇らしいなどの気持ちも扱います。

## 開き方

GitHub Pagesの公開元は `/docs` フォルダです。
このフォルダで、次のどちらかの方法で確認できます。

### GitHub Pagesで見る

Pagesを有効にしたあと、次のURLで開けます。

[https://aco-chan.github.io/vortex/](https://aco-chan.github.io/vortex/)

### そのまま開く

`docs/index.html` をダブルクリックするとブラウザで開けます。

### ローカル表示で開く

表示がうまくいかない場合は、ターミナルでこのフォルダに移動してから次を実行します。

```bash
python3 -m http.server 8000 -d docs
```

そのあと、ブラウザで次を開きます。

```text
http://localhost:8000
```

## ファイル

- `docs/index.html`: 画面の骨組み
- `docs/styles.css`: 見た目、余白、色、絵本風の飾り
- `docs/script.js`: 選択肢、画面の切り替え、物語の仮データ
- `CHANGELOG.md`: 変更履歴とバージョン管理
- `push-docs.command`: 変更をGitHubへ送るためのダブルクリック用ファイル

## 変更履歴

変更内容は `CHANGELOG.md` で管理します。
`push-docs.command` を使うと、READMEや `/docs` のファイルと一緒に `CHANGELOG.md` もGitHubへ送られます。

画面表示や機能を変更した場合は、APP_VERSIONを更新すること。
