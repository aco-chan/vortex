# きもちの絵本

感情を選んでいくと、その気持ちを否定せずに受けとめる、絵本風の短い物語を表示するiPhone向けWebモックです。

## 入っているもの

- タイトル画面
- 感情選択画面
- 気持ちの深掘り選択画面
- 物語の景色選択画面
- 絵本風ストーリー出力画面

## 開き方

このフォルダで、次のどちらかの方法で確認できます。

### GitHub Pagesで見る

Pagesを有効にしたあと、次のURLで開けます。

```text
https://aco-chan.github.io/vortex/
```

### そのまま開く

`index.html` をダブルクリックするとブラウザで開けます。

### ローカル表示で開く

表示がうまくいかない場合は、ターミナルでこのフォルダに移動してから次を実行します。

```bash
python3 -m http.server 8000
```

そのあと、ブラウザで次を開きます。

```text
http://localhost:8000
```

## ファイル

- `index.html`: 画面の骨組み
- `styles.css`: 見た目、余白、色、絵本風の飾り
- `script.js`: 選択肢、画面の切り替え、物語の仮データ
