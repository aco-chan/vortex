# Changelog

このファイルは、きもちの絵本の変更履歴です。
最新の変更が一番上に来ます。

## v0.6.6 - 2026-07-12

- 物語の最後の「〜できる子なんですね」という評価的な締め方をやめました。
- 「見つけた ちから」も、主人公を判定する文ではなく、物語の中で残った気持ちや小さな変化を描く文に寄せました。

## v0.6.5 - 2026-07-12

- 画面表示用のバージョン番号を更新しました。
- READMEに、画面表示や機能を変更した場合は `APP_VERSION` を更新する作業ルールを追加しました。

## v0.6.4 - 2026-07-12

- 既存の感情を6グループに分け、グループごとに別の物語構造で生成するようにしました。
- 汎用テンプレートで使っていた共通フレーズを削除し、導入、身体描写、会話、結び、コンパスをグループ別にしました。
- 代表6ケースが別々の絵本として読めるように、怒り、悲しみ、不安、疲れ、喜び、達成の流れを分けました。

## v0.6.3 - 2026-07-12

- 感情選択の前に、きょう起きた出来事を短く入力できる画面を追加しました。
- 入力された出来事を `state.eventText` に保存し、物語の場面として自然に反映するようにしました。
- 出来事入力は任意にし、戻っても内容が残り、さいしょからの時だけ消えるようにしました。

## v0.6.2 - 2026-07-11

- 画面に表示するバージョン名を `docs/script.js` の `APP_VERSION` だけで管理するようにしました。
- HTML内のバージョン直書きを削除し、JavaScriptから表示するようにしました。

## v0.6.1 - 2026-07-11

- 入力フローから「景色」を選ぶ画面を削除し、名前、いまのきもち、その奥のきもちだけで絵本を生成できるようにしました。
- 奥のきもちを選んだあと、「おはなしを よんでみる」ボタンから絵本へ進む流れにしました。
- 物語の背景描写は、選ばれたきもちに応じて内部で自然に決まるようにしました。

## v0.6.0 - 2026-07-11

- 物語生成を短い固定文の組み合わせから、原作ソースを参考にした物語タイプ方式へ変更しました。
- 指定された8つの優先ケースで、日常場面、会話、小さな変化、主人公の力が出る絵本風の文章を表示するようにしました。
- すべての感情と奥の気持ちに、自然なフォールバック物語が出るようにしました。
- 物語本文の主人公表現に、入力した名前と呼び方を使う処理を維持しました。

## v0.5.3 - 2026-07-11

- 物語本文で主人公を指す表現に、入力された名前と呼び方を必ず使うようにしました。
- 名前未入力時の物語本文に「あなた」が出ないよう、内部表示名を `しゅじんこう` に変更しました。
- ひらがな化の過程で崩れていた一部の文言を修正しました。

## v0.5.2 - 2026-07-11

- UI内の日本語を、子どもが読みやすいようにひらがな中心へ調整しました。
- 感情を「きもち」と表現し、画面文言を「えらんでね」などのやさしい言い方に寄せました。
- 感情・奥の気持ち・物語本文・きょうのコンパスの漢字量を減らしました。

## v0.5.1 - 2026-07-11

- GitHub Pages上で古い `script.js` / `styles.css` がキャッシュされにくいよう、読み込みURLにバージョンを付けました。
- 外部JavaScriptが古い場合でも、「はじめる」ボタンから名前入力画面へ進める補助処理を追加しました。
- 公開ページ上で読み込まれているJavaScriptの確認用に `window.EmoCompassVersion` を追加しました。

## v0.5.0 - 2026-07-07

- 最初に主人公の名前を入力できる画面を追加しました。
- 呼び方を「ちゃん」「くん」「さん」「呼び捨て」から選べるようにしました。
- ネガティブ寄りだけでなく、うれしい、楽しい、誇らしい、安心などの感情も選べるようにしました。
- 感情の奥にある気持ちの選択肢を、しずかな気持ち・つよい気持ち・あたたかい気持ち・ひらいていく気持ちに対応して増やしました。
- 絵本風ストーリーに入力した名前、ぴー、ごまじろう、今日のコンパスを反映するようにしました。

## v0.4.0 - 2026-07-07

- トップ画面にキャッシュ確認用のバージョン表示を追加しました。
- `CHANGELOG.md` を追加し、今後の変更履歴をルートで管理するようにしました。
- `push-docs.command` で `CHANGELOG.md` も一緒にGitHubへ送れるようにしました。

## v0.3.0 - 2026-07-07

- 論文の `R -> L -> T -> R'` の考え方を画面構成と物語生成に反映しました。
- 絵本素材を参考に、ぴーとごまじろうが感情の奥にある願いを見つける仕様にしました。
- 物語の最後に「新しい見方」を表示するようにしました。

## v0.2.0 - 2026-07-07

- GitHub Pagesの公開元を `/docs` にするため、Web本体を `docs/` 配下へ移動しました。
- READMEにGitHub Pagesのリンクと `/docs` 前提の開き方を追加しました。

## v0.1.0 - 2026-07-07

- iPhone向けWebモックを作成しました。
- 感情選択、深掘り選択、景色選択、絵本風ストーリー出力を実装しました。
