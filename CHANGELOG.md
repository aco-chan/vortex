# Changelog

このファイルは、きもちの絵本の変更履歴です。
最新の変更が一番上に来ます。

## v0.4.0 - 2026-07-07

- トップ画面にキャッシュ確認用のバージョン表示を追加しました。
- `CHANGELOG.md` を追加し、今後の変更履歴をルートで管理するようにしました。
- `push-docs.command` で `CHANGELOG.md` も一緒にGitHubへ送れるようにしました。

## v0.3.0 - 2026-07-07

- 論文の `R -> L -> T -> R'` の考え方を画面構成と物語生成に反映しました。
- 絵本素材を参考に、ぴーとごまじろうが感情の奥にある願いを見つける仕様にしました。
- 物語の最後に「新しい見方」を表示するようにしました。

## v0.2.0 - 2026-07-07

- GitHub Pagesの公開元を `/docs` にするため、Web本体を `docs/` 配下へ移動しました。
- READMEにGitHub Pagesのリンクと `/docs` 前提の開き方を追加しました。

## v0.1.0 - 2026-07-07

- iPhone向けWebモックを作成しました。
