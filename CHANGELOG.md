# Changelog

このファイルは、きもちの絵本の変更履歴です。
最新の変更が一番上に来ます。

## v0.5.2 - 2026-07-11

- UI内の日本語を、子どもが読みやすいようにひらがな中心へ調整しました。
- 感情を「きもち」と表現し、画面文言を「えらんでね」などのやさしい言い方に寄せました。
- 感情・奥の気持ち・物語本文・きょうのコンパスの漢字量を減らしました。

## v0.5.1 - 2026-07-11

- GitHub Pages上で古い `script.js` / `styles.css` がキャッシュされにくいよう、読み込みURLにバージョンを付けました。
- 外部JavaScriptが古い場合でも、「はじめる」ボタンから名前入力画面へ進める補助処理を追加しました。
- 公開ページ上で読み込まれているJavaScriptの確認用に `window.EmoCompassVersion` を追加しました。

## v0.5.0 - 2026-07-07

- 最初に主人公の名前を入力できる画面を追加しました。
- 呼び方を「ちゃん」「くん」「さん」「呼び捨て」から選べるようにしました。
- ネガティブ寄りだけでなく、うれしい、楽しい、誇らしい、安心などの感情も選べるようにしました。
- 感情の奥にある気持ちの選択肢を、しずかな気持ち・つよい気持ち・あたたかい気持ち・ひらいていく気持ちに対応して増やしました。
- 絵本風ストーリーに入力した名前、ぴー、ごまじろう、今日のコンパスを反映するようにしました。

## v0.4.0 - 2026-07-07

- トップ画面にキャッシュ確認用のバージョン表示を追加しました。
- `CHANGELOG.md` を追加し、今後の変更履歴をルートで管理するようにしました。
- `push-docs.command` で `CHANGELOG.md` も一緒にGitHubへ送れるようにしました。

## v0.3.0 - 2026-07-07

- 論文の `R -> L -> T -> R'` の考え方を画面構成と物語生成に反映しました。
- 絵本素材を参考に、ぴーとごまじろうが感情の奥にある願いを見つける仕様にしました。
- 物語の最後に「新しい見方」を表示するようにしました。

## v0.2.0 - 2026-07-07

- GitHub Pagesの公開元を `/docs` にするため、Web本体を `docs/` 配下へ移動しました。
- READMEにGitHub Pagesのリンクと `/docs` 前提の開き方を追加しました。

## v0.1.0 - 2026-07-07

- iPhone向けWebモックを作成しました。
- 感情選択、深掘り選択、景色選択、絵本風ストーリー出力を実装しました。
